# myfinance

A modern, fun personal-finance web app.

- **Stack:** SvelteKit + TypeScript + Tailwind (neobrutalist)
- **DB:** Turso (libSQL) + Drizzle ORM
- **Auth:** Auth.js (GitHub + optional Google)
- **Notifications:** Browser Web Push + in-app toasts
- **Currency:** Multi-currency with Frankfurter.app FX rates
- **Hosting:** Vercel (primary); Cloudflare Pages + Netlify supported

## Features

- Record income and expenses across multiple accounts and currencies
- Recurring income and expense rules with auto-materialization
- Category management with seeded defaults
- Budgets with threshold alerts
- Monthly reports with charts
- CSV / JSON export; CSV import with column mapping
- Web Push reminders for upcoming recurring bills and budget alerts
- Account deletion with full GDPR cascade
